# HoosNotes

HoosNotes is web application that allows users to upload notes, keep track of tasks on a calendar/to-do list, and chat with other users. The application is built on Django.

## Usage
To use the application, visit [https://projecta20.herokuapp.com/](https://projecta20.herokuapp.com/) and log in with your Google account.

## Important Note

HoosNotes is no longer under development. As such, certain features may be deprecated and/or not work properly.
